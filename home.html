This page contains web applications to help with the creation of the JSON meta-data parameter files for dcmqi. Please read the documentation of dcmqi for usage details. dcmqi (DICOM for Quantitative Imaging) is a free, open source library that implements conversion of the data stored in commonly used research formats into the standard DICOM representation. dcmqi is being developed by the QIICR project, which is supported by NIH National Cancer Institute, award U24 CA180918.
